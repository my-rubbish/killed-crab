# 螃蟹打击桌宠 - 贴图清单

## 画布尺寸
- 所有贴图尺寸：192x128 像素
- 格式：PNG（支持透明通道）
- 风格：像素画风

## 需要的贴图文件

### 1. 螃蟹身体状态（5张）
- `crab_healthy.png` - 健康状态的螃蟹身体
- `crab_cracked.png` - 壳出现裂缝的螃蟹身体
- `crab_damaged.png` - 严重损坏的螃蟹身体
- `crab_dying.png` - 垂死状态的螃蟹身体（暗淡）
- `crab_dead.png` - 死亡状态的螃蟹身体（最暗淡）

### 2. 螃蟹眼神（5张）
- `eyes_threatening.png` - 威胁性眼神（健康时）
- `eyes_weakening.png` - 力不从心眼神（受损时）
- `eyes_dying.png` - 垂死眼神
- `eyes_dead.png` - 死亡眼神
- `eyes_clicked.png` - 被点击时的表情 (><)

### 3. 螃蟹部位（6张）
- `left_claw.png` - 左钳子
- `right_claw.png` - 右钳子
- `left_leg1.png` - 左腿1
- `left_leg2.png` - 左腿2
- `right_leg1.png` - 右腿1
- `right_leg2.png` - 右腿2

### 4. 特效（1张）
- `debris.png` - 碎片特效（小碎片）

## 图层说明

贴图在游戏中的显示顺序（从下到上）：
1. 螃蟹身体（底层）
2. 螃蟹腿部（4条腿）
3. 螃蟹钳子（2个钳子）
4. 螃蟹眼神（顶层）

## 部位掉落时机

- 8次点击：左腿1掉落
- 12次点击：右腿1掉落
- 18次点击：左腿2掉落
- 22次点击：右腿2掉落
- 28次点击：左钳子掉落
- 32次点击：右钳子掉落

## 状态变化时机

- 5次点击：出现裂缝
- 15次点击：严重损坏
- 25次点击：垂死状态
- 35次点击：死亡状态

## 注意事项

1. 所有贴图必须有透明背景
2. 像素画风格，避免抗锯齿
3. 部位贴图要与身体贴图对齐
4. 眼神贴图只包含眼部，其他部分透明
5. 碎片可以是小方块或小三角形

## 文件放置位置

将所有贴图文件放在 `textures/` 文件夹中。
