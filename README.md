# 螃蟹打击桌宠游戏

一个简单的解压小游戏，玩家通过点击螃蟹来释放压力。

## 游戏特色

- 🦀 可爱的像素风螃蟹
- 🖱️ 简单的鼠标点击操作
- 💥 震动和碎片特效
- 📱 桌宠形式（无边框透明窗口）
- 🎯 渐进式破坏系统

## 游戏玩法

1. **点击螃蟹**：用鼠标左键点击螃蟹
2. **观察变化**：螃蟹会随着点击次数发生变化
3. **部位掉落**：螃蟹的腿和钳子会逐渐掉落
4. **状态变化**：螃蟹的外观和眼神会变化
5. **特效展示**：每次点击都有震动和碎片效果

## 破坏阶段

### 状态变化
- **0-4次点击**：健康状态，威胁性眼神
- **5-14次点击**：壳出现裂缝
- **15-24次点击**：严重损坏，眼神力不从心
- **25-34次点击**：垂死状态，眼神暗淡
- **35次以上**：死亡状态，完全暗淡

### 部位掉落时机
- **8次点击**：左腿1掉落
- **12次点击**：右腿1掉落
- **18次点击**：左腿2掉落
- **22次点击**：右腿2掉落
- **28次点击**：左钳子掉落
- **32次点击**：右钳子掉落

## 技术特色

- **桌宠模式**：无边框透明窗口，可作为桌面宠物
- **像素艺术**：192x128分辨率的精美像素画
- **动态特效**：震动、碎片飞散、部位掉落动画
- **状态系统**：5种螃蟹状态和4种眼神状态

## 下一步工作

1. **准备贴图**：根据 `textures/贴图清单.md` 制作所需的像素画贴图
2. **导入贴图**：将贴图文件放入 `textures/` 文件夹
3. **配置贴图**：在脚本中启用贴图加载代码
4. **测试游戏**：在Godot中运行项目
5. **添加音效**：为点击、部位掉落等添加音效
6. **打包发布**：导出为可执行文件
